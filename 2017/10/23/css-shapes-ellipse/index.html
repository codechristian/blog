CSS Shapes: Ellipse

October 23, 2017

CSS Shapes

CSS shapes allow floated images to have text flow around it based on the shape of the image.

First lets start with some simple HTML: 


 <h1>Duckrs The Duck</h1>
 <img src=“duck.jpg” alt=“Duckrs The Duck”>
 <p>Duckrs The Duck lives in a beautiful pond where he enjoys long swims and the peaceful sounds of nature.</p>

Heres what the flow of text would look like without CSS shapes: 


 img {
     float: left;
     margin-right: 2em;
     margin-bottom: 0.5em;
     border-radius: 50%;
 }

With CSS shapes we can manipulate how the text flows around the image. Here we are using the ellipse property : 


 img {
     float: left;
     margin-right: 2em;
     margin-bottom: 0.5em;
     border-radius: 50%;
     shape-outside: ellipse(45% 45%);
 }
